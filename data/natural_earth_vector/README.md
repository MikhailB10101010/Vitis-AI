## Доступные слои Natural Earth Data

В проекте используются данные из библиотеки **Natural Earth** (версия 4.1.0). Ниже приведен полный список доступных слоев, сгруппированных по тематике и масштабу.

Доступны три уровня детализации:
*   **10m** (1:10,000,000) — максимальная детализация.
*   **50m** (1:50,000,000) — средняя детализация.
*   **110m** (1:110,000,000) — низкая детализация (для генерализованных карт).

### Примечания по использованию

1.  **Приоритет слоев:** Для отрисовки карты рекомендуется сначала добавлять полигональные слои (`land`, `ocean`, `countries`, `lakes`), затем линейные (`coastline`, `rivers`, `boundary_lines`) и поверх них точечные (`populated_places`, `airports`).
2.  **Выбор масштаба:** Для создания детальных карт регионов используйте слои с префиксом `ne_10m_`. Для глобальных карт мира или обзорных карт используйте `ne_110m_` для ускорения рендеринга.
3.  **Спорные территории:** Обратите внимание на наличие слоев `disputed_areas` и версий `_chn` (для китайской спецификации границ), если требуется отображение в соответствии с законодательством определенных стран.
